为什么要推荐中国能用的VPN这么难？如此饱和的VPN翻墙软件市场，也难找到几款好用的产品？

现在适合中国的VPN加起来恐怕连10款都不到。但读者也别气馁，好消息是，尽管每天都有众多翻墙软件失效，每天也有很多翻墙工具上线。而且重视中国市场的服务商们，一直孜孜不倦地改进着算法，每天增加着更多的服务器，每隔几周都换掉镜像域名，为的就是让你能访问到真正的国际互联网。
